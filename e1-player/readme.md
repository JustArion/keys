### Requirements:
- Node.js (v20.X)
- pnpm
    - (For Windows)
      - Choco: `choco install pnpm`
      - Scoop: `scoop install pnpm`
      - npm: `npm install -g pnpm@latest-10`
    - (For Linux)
      - npm: `npm install -g pnpm@latest-10`

### Install & Run 
```
pnpm install
npx ts-node src/index.ts
```

### Working On:
- Windows (x86_64)
- Linux (x86_64)


- Testing on Linux/aarch64 `failed` for me.
- Untested on Mac
